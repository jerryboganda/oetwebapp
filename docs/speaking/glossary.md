# Speaking Module — Glossary

| Term | Meaning |
|------|---------|
| **Candidate** | The learner taking the OET Speaking test. |
| **Interlocutor** | Trained person who plays the patient / carer / relative / colleague in the role-play. Does NOT score the candidate. |
| **Role-play** | The ~5-minute assessed task. Two per OET Speaking test. |
| **Role-play card** | The scenario brief. Candidate version (learner-facing) + interlocutor version (hidden script). |
| **Warm-up** | Short pre-test conversation. Unscored. Helps the candidate settle. |
| **Prep / Preparation time** | 3 minutes per role-play to read the card, plan, ask about unfamiliar words. |
| **ICE** | Ideas, Concerns, Expectations — the patient perspective the candidate must elicit. |
| **Signposting** | Verbal markers that organise an explanation ("First, I'll explain… Then we'll discuss…"). |
| **Lay language** | Patient-friendly translation of clinical jargon. |
| **Linguistic criteria** | Intelligibility, Fluency, Appropriateness, Resources of Grammar and Expression. |
| **Clinical communication criteria** | Relationship building, Understanding patient perspective, Providing structure, Information gathering, Information giving. |
| **Readiness band** | Platform's estimated grade (Not ready / Developing / Borderline / Exam-ready / Strong). NOT an official OET score. |
| **Mock set** | Two paired role-plays delivered as a single mock exam. |
| **Bridge** | Short handoff between RP1 and RP2 in a mock. |
| **Calibration sample** | A finished session promoted to gold-marked status for tutor inter-rater drift tracking. |
| **Drift** | Mean absolute error between a tutor's scoring and the gold scoring. |
| **Drill** | Single-focus micro-practice item targeting one or two criteria. |
| **Egress** | LiveKit-side recording pipeline writing track-composite audio/video to S3. |
| **Cue** | Tutor-side button in the interlocutor script that emits a prompt event to the learner. |
| **Persona prompt** | The system-prompt block that tells the AI patient how to behave. Cached. |
| **Evidence quote** | A verbatim transcript substring the AI must cite to justify a score. Verified server-side. |
| **Profession** | The candidate's healthcare role (nursing, medicine, pharmacy, physio, etc.). Drives card visibility. |
