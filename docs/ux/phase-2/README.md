# Phase 2 — Learner JTBD & Journey Maps

Produce Figma-ready JTBD statements, journey maps, and flow specs for Nurse Nadia (P1), Doctor Dinesh (P2), Student Sana (P3).

## Required artifacts
For each persona:
1. `learner-<persona>-jtbd.md` — job statement, current incumbent, pain points, success metric.
2. `learner-<persona>-journey.md` — stages Awareness → Exploration → Action → Outcome → Retention. Record think/feel/do + pain points + opportunities.
3. `learner-flow-<flow>.md` — entry, steps, decision points, exits. Include a11y requirements + 360/768/1280 notes.

- Every T0 learner route has a linked JTBD step in at least one journey.
