# Recalls Data Entry Execution Packet

This folder contains the operator-ready packet for entering Recalls vocabulary data with source traceability, structure preservation, and zero invented content.

## Files

- `PRD.md` - product and operations requirements for the data-entry job.
- `PROGRESS.md` - Ralph-style loop tracker for research, artifacts, validation, and review.
- `source-register.template.csv` - source-document inventory template.
- `master-manifest.template.csv` - row-level evidence and import manifest template.
- `validation-checklist.md` - pre-import, import, and post-import gates.
- `reviewer-signoff-workflow.md` - statuses, roles, handoffs, and approval rules.
- `system-hardening-backlog.md` - engineering backlog before large production entry.

## Non-Negotiable Rule

No source, no quote, no reviewer sign-off, no learner visibility.
