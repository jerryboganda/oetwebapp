## Recommended Reading Order

1. Start with the master manual.
2. Read the learner, expert, and admin manuals based on role or area of ownership.
3. Use the cross-system manual to understand dependencies and lifecycle flows.
4. Use the audit fact base when you need route, status, or evidence-oriented detail behind the final write-up.
