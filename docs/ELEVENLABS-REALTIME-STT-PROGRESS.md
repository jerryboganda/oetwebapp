## Current Status

Implementation is in the protected real-provider hardening slice. The learner UI now prefers browser `AudioWorklet` PCM 16 kHz mono realtime chunks while retaining `MediaRecorder` as the batch fallback recording. The experimental server-side ElevenLabs websocket adapter remains behind real-provider gates: explicit readiness flag, topology/region declaration, positive estimated pricing, adult/managed-learner policy flags, conservative daily/monthly budget checks, and protected smoke secret gating. Mock realtime STT is lifecycle/caption-only for durable learner transcripts; persisted learner turns still use batch ASR unless a non-mock provider returns a final transcript.
