# Phase 5 — Content & Copy Audit

## Artifacts
- `voice-and-tone.md` — canonical voice per portal (learner = warm-expert; expert = precise-clinical; admin = neutral-ops; sponsor = professional-B2B).
- `copy-rewrites.md` — before/after for every flagged string from Phase 1 content inventory.
- `empty-error-loading-copy.md` — canonical state copy (e.g., "No results yet — try widening your filters").
- `microcopy-glossary.md` — OET terms: "scaled score", "raw score", "350/500", "CBLA", "sub-test", "profession-specific".

## Rules
- No developer hints in user-visible copy.
- No "coming soon" without a date or CTA.
- Every error has: what happened · why · what to do next.
- Every CTA is a verb + object ("Start mock", "Submit for review").
- Numbers and thresholds always use the scoring utilities per AGENTS.md.

## Exit criteria
- Zero dev-copy strings remaining in T0/T1 routes.
- Every empty/loading/error state mapped to canonical copy.
